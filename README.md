# ( xom669 | Dipanjan ) Portfolio

A brutalist, comic-inspired sketchbook portfolio for Dipanjan Baidya. Built with React, Vite, Tailwind CSS, Motion, and Supabase.

## 🎨 Features
- **Brutalist Comic Aesthetic**: Stark ink borders, halftone patterns, and vibrant pop-art colors.
- **Dynamic Projects Section**: Interactive bento grid showing your latest work.
- **Monster Shooter Widget**: A retro FPS-style animation with rotating motivational quotes.
- **Admin Panel**: A custom CMS interface (built to connect with Supabase) to manage your projects.
- **Fully Responsive**: Designed for all screen sizes with a mobile-first approach.

## 🛠️ Tech Stack
- **Frontend**: React 19, Vite, Tailwind CSS 4.
- **Animations**: Motion (formerly Framer Motion).
- **Backend/DB**: Supabase (PostgreSQL).
- **Icons**: Lucide React & Material Symbols.

## 🚀 Deployment (Vercel)
1. **Push to GitHub**: Use the export tool in AI Studio.
2. **Connect to Vercel**: Import the repository.
3. **Set Environment Variables**:
   - `VITE_SUPABASE_URL`: Your Supabase URL.
   - `VITE_SUPABASE_ANON_KEY`: Your Supabase API Key.
4. **Setup Database**: Use the SQL provided in the "Supabase Setup" section of the agent chat.
